API 명세서

프로젝트명: Next.js Supabase 게시판 프로젝트

작성일: 2026-01-20

Base URL: Supabase API를 직접 사용

1. 인증 API

POST 회원가입

Endpoint: auth/v1/signup

설명: 신규 사용자 회원가입

Request Body

{ "email": "user@example.com", "password": "password123" }

Response (Success)

{ "user": { "id": "uuid", "email": "user@example.com" }, "session": { "access_token": "string", "refresh_token": "string" } }

POST 로그인

Endpoint: auth/v1/token?grant_type=password

설명: 사용자 로그인

Request Body

{ "email": "user@example.com", "password": "password123" }

Response (Success)

{ "access_token": "string", "refresh_token": "string", "user": { "id": "uuid", "email": "user@example.com" } }

POST 로그아웃

Endpoint: auth/v1/logout

설명: 사용자 로그아웃

Headers: Authorization: Bearer {access_token}

2. 게시판 API

GET 게시글 목록 조회

Endpoint: rest/v1/posts

설명: 게시글 목록을 페이지네이션과 함께 조회

Query Parameters

파라미터 타입 필수 설명
page number N 페이지 번호 (기본값: 1)
limit number N 페이지당 항목 수 (기본값: 10)
order string N 정렬 기준 (created_at.desc)

Response (Success)

[ { "id": "uuid", "user_id": "uuid", "title": "게시글 제목", "content": "게시글 내용", "views": 0, "created_at": "2026-01-20T00:00:00Z", "updated_at": "2026-01-20T00:00:00Z", "profiles": { "username": "사용자명" } } ]

GET 게시글 상세 조회

Endpoint: rest/v1/posts?id=eq.{id}

설명: 특정 게시글의 상세 정보 조회

Response (Success)

{ "id": "uuid", "user_id": "uuid", "title": "게시글 제목", "content": "게시글 내용", "views": 1, "created_at": "2026-01-20T00:00:00Z", "updated_at": "2026-01-20T00:00:00Z", "profiles": { "username": "사용자명", "avatar_url": "url" } }

POST 게시글 작성

Endpoint: rest/v1/posts

설명: 새 게시글 작성 (인증 필요)

Headers: Authorization: Bearer {access_token}

Request Body

{ "title": "게시글 제목", "content": "게시글 내용" }

PUT 게시글 수정

Endpoint: rest/v1/posts?id=eq.{id}

설명: 게시글 수정 (본인만 가능)

Headers: Authorization: Bearer {access_token}

Request Body

{ "title": "수정된 제목", "content": "수정된 내용" }

DELETE 게시글 삭제

Endpoint: rest/v1/posts?id=eq.{id}

설명: 게시글 삭제 (본인만 가능)

Headers: Authorization: Bearer {access_token}

3. 댓글 API

GET 댓글 목록 조회

Endpoint: rest/v1/comments?post_id=eq.{post_id}

설명: 특정 게시글의 댓글 목록 조회

POST 댓글 작성

Endpoint: rest/v1/comments

설명: 댓글 작성 (인증 필요)

Headers: Authorization: Bearer {access_token}

Request Body

{ "post_id": "uuid", "content": "댓글 내용" }

4. 에러 응답

상태 코드 설명
400 잘못된 요청
401 인증 실패
403 권한 없음
404 리소스를 찾을 수 없음
500 서버 오류
